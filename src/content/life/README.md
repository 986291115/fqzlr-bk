# 生活模块

把每天的健康、日记、想法、规划和足迹统一记录。

## 功能导航

| 功能 | 路径 | 说明 |
|------|------|------|
| 健康 | `/life/health/` | 运动、睡眠、饮食记录 |
| 笔记本 | `/life/notebooks/` | 成长日记、学习复盘、灵感随笔 |
| 奇思妙想 | `/life/ideas/` | 随手记录的灵感与创意 |
| 规划打卡 | `/life/plans/` | 每日习惯目标与打卡统计 |
| 去过的地方 | `/life/places/` | 足迹地图与旅行记录 |

## 数据存储结构

```
src/content/life/
├── meta/                   # 生活模块元数据配置
│   └── README.md           # 生活模块说明（本文件）
├── health/                 # 健康记录
│   ├── README.md           # 健康功能说明
│   ├── workout/            # 运动记录
│   ├── sleep/             # 睡眠记录
│   └── food/              # 饮食记录
├── ideas/                  # 奇思妙想
├── notebooks/             # 笔记本
│   ├── growth/            # 成长日记
│   ├── study/             # 学习复盘
│   └── ideas/             # 灵感随笔
├── places/                # 去过的地方
├── plans/                 # 规划打卡
└── README.md              # 生活模块总览说明
```

## 页面展示

生活首页展示以下内容：

- **目标完成度**：环形进度图显示本月打卡、年运动次数、灵感完成率、连续打卡
- **数据对比**：同比/环比数据对比卡片
- **趋势数据**：近8周跑步里程和平均睡眠折线图
- **打卡活跃度**：热力图展示全年打卡情况
- **详细统计**：运动、睡眠、饮食、灵感、地点等详细数据
