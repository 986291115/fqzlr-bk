# 规划打卡

管理每日习惯目标，自动统计打卡完成情况。

## 数据存放

`src/content/life/plans/`

## 记录格式

在 `plans/` 下新建 md 文件：

```yaml
---
planName: "晨跑计划"          # 计划名称（必填）
targetDesc: "每天至少跑步 5 公里"  # 目标描述（选填）
dailyTarget: 1               # 每日目标次数（选填，默认1）
monthlyTarget: 20            # 月度目标次数（选填，默认20）
checkins:                   # 打卡记录，格式为 YYYY-MM-DD
  - "2026-04-01"
  - "2026-04-03"
  - "2026-04-05"
---
```

## 字段说明

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| planName | 是 | 计划名称 | 晨跑计划 |
| targetDesc | 否 | 目标详细描述 | 每天至少跑步 5 公里 |
| dailyTarget | 否 | 每日目标次数 | 1 |
| monthlyTarget | 否 | 月度目标次数 | 20 |
| checkins | 否 | 打卡日期数组 | ["2026-04-01", "2026-04-03"] |

## 数据展示

- 页面顶部显示日打卡完成度、月打卡完成度统计
- 规划列表按完成进度排序，显示本月进度条
- 每个规划卡片显示累计打卡次数和今年打卡数
