# 奇思妙想

随手记录灵感、短句和可执行的创意想法。

## 数据存放

`src/content/life/ideas/`

## 记录格式

在 `ideas/` 下新建 md 文件：

```yaml
---
status: "todo"              # 状态：done（已完成）或 todo（待完成）
content: "每周三次力量训练习惯"  # 想法内容（必填）
createdAt: "2026-03-01"     # 创建日期（选填）
completedAt: "2026-04-10"   # 完成日期，status 为 done 时填写（选填）
label: "健身"               # 标签（选填）
---
```

## 字段说明

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| status | 是 | 状态 | `done` / `todo` |
| content | 是 | 想法内容 | 每周三次力量训练习惯 |
| createdAt | 否 | 记录创建日期 | 2026-03-01 |
| completedAt | 否 | 完成日期 | 2026-04-10 |
| label | 否 | 标签分类 | 健身 |

## 状态说明

- `todo`：待完成的想法，会显示在「待完成」区域
- `done`：已完成的想法，会显示在「已完成」区域并按完成时间倒序

## 数据展示

- 页面顶部显示总想法数、已完成数、待完成数、本月完成数
- 已完成区域显示绿色边框
- 待完成区域显示黄色边框
