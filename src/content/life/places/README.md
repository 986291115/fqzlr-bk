# 去过的地方

足迹地图与旅行记录，地图仅展示中国，去过次数越多颜色越深。

## 数据存放

`src/content/life/places/`

## 记录格式

在 `places/` 下新建 md 文件：

```yaml
---
date: "2025-06-18"          # 到访日期（选填）
province: "北京"            # 省份（必填）
city: "北京"                # 城市（选填）
experience: "逛了故宫和国博，步行很多但很充实。"  # 体验记录（选填）
visitCount: 2               # 到访次数（选填，默认1）
---
```

## 字段说明

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| province | 是 | 省份名称 | 北京 |
| city | 否 | 城市名称 | 北京 |
| date | 否 | 到访日期 | 2025-06-18 |
| experience | 否 | 旅行体验 | 逛了故宫和国博 |
| visitCount | 否 | 到访次数 | 2 |

## 数据展示

- 页面顶部显示去过省份数、总到访次数、今年出行数、总地点数
- 中国地图热力图展示足迹，去过次数越多颜色越深
- 地点列表按日期倒序显示所有记录
