## 不会代码也能改博客

很多人搭好博客之后想改点东西——换个颜色、加个功能、调一下布局——但一看代码就懵了。这期教你用 AI 工具来干这事，你只需要用中文描述想要什么效果，AI 帮你改代码。

这里用两款国产工具：**阿里 Qoder** 和 **字节 Trae**。都是免费的，对新手很友好。

## 工具介绍

### 阿里 Qoder

[Qoder](https://qoder.com.cn) 是阿里推出的 AI 编程助手，擅长整体性的代码生成和重构。你可以把它理解为一个能读懂你项目代码的 AI 助手，告诉它要改什么，它直接帮你改好。

适合场景：整体重构、添加新功能模块、大范围样式调整。

### 字节 Trae

[Trae](https://www.trae.cn/ide/download) 是字节跳动做的 AI IDE（代码编辑器），内置了 AI 辅助编程能力。它更擅长细节优化和错误修复，可以一边写代码一边让 AI 帮忙。

适合场景：细节调整、修 bug、局部优化。

### 对比

| | Qoder | Trae |
|---|-------|------|
| 开发商 | 阿里巴巴 | 字节跳动 |
| 擅长 | 整体重构、功能添加 | 细节优化、错误修复 |
| 使用方式 | 描述需求 -> 生成方案 | 编辑器内实时辅助 |
| 上手难度 | 简单 | 简单 |

两个都可以试试，看哪个更合你习惯。

## 操作流程

整个流程其实很简单：

### 1. 安装工具

去官网下载安装就行，Qoder 和 Trae 都有 Windows 版本。

### 2. 导入博客项目

把你在 GitHub 上的博客仓库克隆到本地，然后用工具打开项目文件夹。

### 3. 描述你的需求

直接用中文告诉 AI 你想改什么。比如：

- "把首页的文章卡片改成圆角阴影样式"
- "在导航栏加一个关于我的页面链接"
- "把代码块的主题换成深色风格"
- "给文章页加一个目录导航"

描述得越具体，AI 改出来的效果越接近你的预期。

### 4. AI 生成方案并应用

AI 会分析你的项目代码，生成修改方案。你确认后它直接改文件。

### 5. 预览效果

本地跑 `pnpm dev` 看效果，不满意就继续让 AI 调。

### 6. 满意后推送

```bash
git add .
git commit -m "修改xxx样式"
git push
```

## 常见修改场景

这些是 AI 工具比较擅长的：

**样式调整**：颜色、字体、间距、圆角、阴影这些视觉相关的改动，AI 处理得很好。

**添加功能**：搜索框、评论区、分享按钮、暗色模式切换等常见功能，基本一句话就能加上。

**响应式适配**：让博客在手机上也好看，AI 能帮你写媒体查询。

**SEO 优化**：让 AI 帮你优化 meta 标签、文章结构。

## 注意事项

**改之前先备份。** 虽然 Git 本身就有版本记录，但保险起见，大改之前先 commit 一次当前状态。

**小步测试。** 不要一次让 AI 改太多东西，改一点测一点，出问题容易定位。

**了解基本概念。** 虽然不用会写代码，但知道 HTML 是结构、CSS 是样式这种基本概念，和 AI 沟通会更顺畅。
